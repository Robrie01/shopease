Payment, Promos & Vouchers

Help with payment methods, discount codes and gift vouchers.

What payment methods do you accept? ›

We accept the following payment methods:

  • Visa
  • Mastercard
  • American Express (Amex)

All payments are processed securely. Card details are never stored on ShopEase servers and are handled by our PCI DSS Level 1 payment provider.

How do I use a promo code? ›

You can enter a promo code at checkout in the Promo Code field. Click Apply to activate the discount. Only one promo code can be used per order.

Current codes: SAVE10 for 10% off, FIRST5 for £5 off your order.

My promo code is not working ›

Please check the following:

  • The code is entered exactly as provided, with no extra spaces
  • The code has not expired
  • The code applies to the items in your cart

If you are still having trouble, contact our support team with the code you are trying to use.

Do you offer gift vouchers? ›

Gift vouchers are not currently available on ShopEase but we plan to introduce them in the future. Keep an eye on the site for updates.

Is my payment information secure? ›

Yes. All transactions are encrypted using 256-bit SSL and processed by a PCI DSS Level 1 certified provider. ShopEase never stores your card details on our servers.
